Dates in Python

Dates in Python are handled by using the package datetime.

Convert a String to a Date

In order to convert a string to a date, you need to use the function strptime. This function takes in two parameters: the string containing the input date and the format of the input date. The output of the function is a datetime object. The format of the output date depends on the locale settings on your computer. The locale is the set of parameters that define your language and region. If your locale is fr_FR, then the format of the output date will be dd/mm/yyyy.

Here is an example:

import datetime

date = datetime.datetime.strptime("November 24, 1980", '%B %d, %Y')
print(date)

date = datetime.datetime.strptime("24 November 1980", '%d %B %Y')
print(date)

date = datetime.datetime.strptime("24/11/1980", '%d/%m/%Y')
print(date)

date = datetime.datetime.strptime("1980/11/24", '%Y/%m/%d')
print(date)
Here is an explanation of the code:
  • Line 3. Converts a date whose format is name of the month day number followed by a comma and the year. The format "%B %d, %Y" indicates precisely this format. %B stands for the name of the month in the language corresponding to the locale settings in your computer. If you want to run this line on a computer where the locale is "fr_FR", you need to change December into DÃ©cembre.
  • Line 6. Same as Line 3, but the format is day month and year without any comma.
  • Line 9. The format of the input date is number of the day/number of the month/year.
  • Line 12. Same as Line 9, but year and day are inverted.
Note that if the input date does not comply with the specified format, a ValueError exception is raised. This is the case of the following line, where the expected format is mm/dd/yyyy but the input date is in the format dd/mm/yyyy. 
date = datetime.datetime.strptime("24/11/1980", '%m/%d/%Y')
The function strptime returns a datetime object that contains the date and also the time. If you just need to get the date, you can call the function date, like in the following example: 
import datetime

date = datetime.datetime.strptime("24/11/1980", '%d/%m/%Y')
# This line prints 1980-11-24 00:00:00 
print(date)
#This line prints 1980-11-24
print(date.date())
