Object-Oriented Programming in Python

Object-oriented Programming (OOP) is a programming paradigm that mandates that the structure of a program be centered around the notion of object. The entities (e.g., persons, geometric figures, numbers) manipulated in a OOP program are represented as objects whose properties and behavior are specified in the definition of classes.

For instance, we can define a class called Cat that describes the properties of any cat (e.g., name and age) and a behavior (e.g., how fast a cat runs) through the definition of functions, a.k.a as methods in the OOP parlance. The class is a model describing what a cat looks like; we can then create an object from the class that describes a particular instance of a cat (the one with name mickey and age 8 that runs slowly).

Defining a Class

Below you'll find the code to define a class Cat in Python with three properties: name, age and speed. 
class Cat:

def __init__(self, name, age, speed):
    self.name = name
    self.age = age
    self.speed = speed

def runs(self):
    if self.speed < 30:
        return "slow"
    else:
        return "fast"

@classmethod
def sound(cls):
    return "Meow"
Here is the explanation of the code:
  • The method __init__ is called the constructor of the class and allows the creation of an instance (i.e., a specific cat) of the class.
  • The method runs is an instance method. This means that it can be called on a specific object of the class. If the method is called on two different objects of the class, the result of the method is likely to be different (two different cats might run at different speed). The parameter self indicates that the method is called on an object.
  • The method sound is a class method. This means that it is called on the class itself, and its result is always the same, for all objects of the class. All cats meow.

Creating Objects

The following code creates two cats and then calls on both the method runs. 
first_cat = Cat("mickey", 10, 40)
second_cat = Cat("mouse", 16, 20)

print("The first cat is", first_cat.runs())
print("The second cat is", second_cat.runs())

print("All cats", Cat.sound())
Here is the explanation of the code:
  • Line 1. Creates a new cat with name mickey, age 10 and speed 40.
  • Line 2. Creates a new cat with name mouse, age 16 and speed 20.
  • Line 4. Prints the string "The first cat is fast".
  • Line 5. Prints the string "The second cat is slow".
  • Line 7. Prints the string "All cats meow".
