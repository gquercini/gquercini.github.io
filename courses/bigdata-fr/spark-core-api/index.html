1. API Spark Core
Big Data — TP 1

L'API Spark Core

Dans ce TP, vous allez coder des fonctions simples en utilisant l’API Spark Core ; vous analyserez les performances de ces fonctions.
Nous allons considérer deux problèmes : le calcul des températures moyennes et la recherche d’amis communs dans un réseau social.

1 Calcul des moyennes

Nous considérons une collection de fichiers CSV contenant des mesures de température au format suivant :

année,mois,jour,heure,minute,seconde,temperature

Vous pouvez trouver les fichiers dans le répertoire hdfs://sar01:9000/data/temperatures/

Voici les détails pour chaque fichier :

  • Le fichier temperatures_86400.csv contient une mesure par jour pour les années 1980 - 2018.
  • Le fichier temperatures_2880.csv contient une mesure toutes les 2880 secondes pour les années 1980 - 2018.
  • Le fichier temperatures_86.csv contient une mesure toutes les 86 secondes pour l’année 1980 uniquement.
  • Le fichier temperatures_10.csv contient une mesure toutes les 10 secondes pour les années 1980 - 2018.

Nous souhaitons implémenter un algorithme Spark pour générer des paires \((y, t_{avg})\), où \(y\) est l’année et \(t_{avg}\) est la température moyenne de cette année.

1.1 Première implémentation

Copiez le fichier /usr/users/dce-admin/vialle/DCE-Spark/template_temperatures.py dans votre répertoire personnel via la commande suivante :

cp /usr/users/dce-admin/vialle/DCE-Spark/template_temperatures.py ./avg_temperatures_slow.py

Ouvrez le fichier avg_temperatures_slow.py.
Le fichier contient l’implémentation de la fonction avg_temperature_slow qui :

  • prend en entrée un RDD, où chaque élément est une ligne d’un fichier texte donné ;
  • retourne un RDD, où chaque élément est une paire clé-valeur (année, temperature moyenne).

Dans ce même fichier, localisez les deux variables input_path et output-path et écrivez le code suivant :

input_path = "hdfs://sar01:9000/data/temperatures/"
output_path = "hdfs://sar01:9000/asi1/asi1_X/"

Remplacez X par le numéro correspondant à votre compte !
N’oubliez pas le / à la fin des chemins de fichiers !

Exécutez les actions suivantes :

  • Lancez le script avg_temperatures_slow.py en utilisant temperatures_86400.csv comme fichier d’entrée.
    Pour ce faire, utilisez la commande suivante :

spark-submit --master spark://sar01:7077 avg_temperatures_slow.py temperatures_86400.csv

  • Vous devriez trouver la sortie du programme dans le dossier suivant :

hdfs://sar01:9000/asi1/asi1_X/temperatures_86400.out

Saisissez la commande suivante pour le vérifier :

hdfs dfs -ls hdfs://sar01:9000/asi1/asi1_X/temperatures_86400.out

  • Si vous souhaitez lire le résultat du calcul, vous pouvez exécuter la commande suivante :

hdfs dfs -cat hdfs://sar01:9000/asi1/asi1_X/temperatures_86400.out/*

2 Amis communs dans un réseau social

Considérons un réseau social représenté par un graphe encodé dans un fichier texte.
Chaque ligne du fichier est une liste d’identifiants séparés par des virgules.
Par exemple, la ligne \(B,D,A\) signifie que \(B\) est ami avec \(D\) et \(A\).
Un extrait du fichier ressemble à ceci :

B,D,A
A,B,C,D
D,A,B,C
C,A,D
...

Nous supposons que la relation d’amitié est symétrique : \((A, B)\) implique \((B, A)\).

Nous voulons obtenir la liste des amis communs pour chaque paire d’individus :

(B, C), [A, D] 
(A, D), [B, C] 
(C, D), [A]
(A, C), [D] 
(B, D), [A] 
(A, B), [D]

Une contrainte supplémentaire : nous ne voulons représenter chaque couple qu’une seule fois et éviter de représenter le couple symétrique.
Autrement dit, si nous produisons \((A, B)\), nous ne voulons pas produire \((B, A)\).

Nous utilisons les fichiers d’entrée suivants (disponibles dans le dossier hdfs://sar01:9000/data/social-network/) :

  • sn_tiny.csv. Petit réseau social, que vous pouvez utiliser pour tester votre implémentation.
  • sn_10k_100k.csv. Réseau social avec \(10^4\) individus et \(10^5\) liens.
  • sn_100k_100k.csv. Réseau social avec \(10^5\) individus et \(10^5\) liens.
  • sn_1k_100k.csv. Réseau social avec \(10^3\) individus et \(10^5\) liens.
  • sn_1m_1m.csv. Réseau social avec \(10^6\) individus et \(10^6\) liens.

2.1 Implémentation

Récupérez le modèle de code avec la commande suivante :

cp /usr/users/dce-admin/vialle/DCE-Spark/template_common_friends.py .
