Requêtes MongoDB

L’objectif de ce tutoriel est d’apprendre à écrire des requêtes en MongoDB.

Préparation de l’environnement

  • Cliquez sur l’icône MongoDB Compass (sur le bureau) pour ouvrir l’application.

  • Dans le menu de gauche, sélectionnez la connexion à Local MongoDB et cliquez sur Connect.

  • Sélectionnez la base de données sakila.

  • Dans la fenêtre à droite, cliquez Open MongoDB shell pour ouvrir le shell MongoDB.

Base de données

La base de données sakila contient des données fictives sur des magasins de location de DVD. La base de données contient les collections suivantes :

  • customer : informations sur les clients.
  • film : informations sur les films.
  • rental : informations sur les locations.
  • staff : informations sur les membres du personnel.
  • store : informations sur les magasins.

Requêtes de base

Exercice

Exercise 1 Ecrivez les requêtes suivantes en MongoDB :

  1. Renvoyez toutes les informations sur tous les clients.

  2. Renvoyez les emails de tous les clients.

  3. Renvoyez les emails des clients des magasins canadiens.

  4. Renvoyez l’identifiant de toutes les locations effectuées par des clients iraniens, où le montant payé est strictement supérieur à 10 dollars.

  5. Renvoyez les prénoms et noms des acteurs qui ont joué dans le film 213.

Opérations sur les tableaux

Exercice

Exercise 2 Ecrivez les requêtes suivantes en MongoDB :

  1. Renvoyez l’identifiant des films qui ont comme caractéristiques spéciales (special features) “Behind the Scenes”.

  2. Renvoyez l’identifiant des films qui ont comme caractéristiques spéciales (special features) toutes les suivantes : “Commentaries” et “Deleted Scenes”.

  3. Renvoyez l’identifiant de tous les films où BURT POSEY a joué un rôle.

  4. Renvoyez l’identifiant du film qui a exactement 15 acteurs.

Pipeline d’agrégation

Exercice

Exercise 3 Écrivez les requêtes suivantes en MongoDB en utilisant des pipelines d’agrégation :

  1. Renvoyez le titre des films loués par TOMMY COLLAZO (pouvez-vous aussi exprimer cette requête avec la fonction find() ?)

  2. Comptez le montant total des paiements de toutes les locations.

  3. Renvoyez le nombre d’acteurs de chaque film.

  4. Triez les films par nombre d’acteurs (ordre décroissant).

  5. Renvoyez le nombre moyen d’acteurs par film.

  6. Renvoyez l’identifiant du client qui a effectué le plus de locations.

  7. Renvoyez le prénom et le nom du client qui a effectué le plus de locations.

  8. Renvoyez le pays où les clients ont loué le plus de films dans la catégorie “Music”.

Opérations de jointure

Exercice

Exercise 4 Ecrivez les requêtes suivantes en MongoDB en utilisant le pipeline d’agrégation :

  1. Renvoyez la langue du film ayant pour titre “ACE GOLDFINGER”.

  2. Renvoyez toutes les informations sur le membre du personnel qui a géré la location numéro 2.
