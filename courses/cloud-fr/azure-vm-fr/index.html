Prise en main de Microsoft Azure

1 Introduction

Nous allons utiliser le portail Azure (une interface utilisateur accessible via un navigateur) pour créer une machine virtuelle Linux exécutant Ubuntu 20.04 LTS. Nous nous connecterons ensuite à la machine virtuelle via SSH et installerons un serveur web NGINX. Enfin, nous utiliserons le navigateur pour nous connecter à ce serveur.

2 Création de la machine virtuelle

  1. Connectez-vous au portail Azure avec vos identifiants CentraleSupélec.

  2. Sélectionnez Machines virtuelles.

  3. Cliquez sur Créer puis sur Machine virtuelle Azure.

  4. Créez un nouveau groupe de ressources et donnez-lui un nom de votre choix.

  5. Donnez un nom à votre machine virtuelle.

  6. Sélectionnez la région (Europe) France Central. Si cette région ne fonctionne pas, essayez-en une autre.

  7. Acceptez les paramètres par défaut pour zone de disponibilité et le type de sécurité.

  8. Sélectionnez Ubuntu Server 20.04 LTS - Generation 2 comme image.

  9. Sélectionnez Standard_B1s - 1 CPU virtuel, 1 Gio de mémoire comme taille, si disponible. Sinon, choisissez la configuration la plus proche ou la plus économique.

  10. Choisissez Clé publique SSH comme type d’authentification.

  11. Choisissez un nom d'utilisateur.

  12. Acceptez les options par défaut pour la clé SSH.

  13. Parmi les ports entrants, sélectionnez SSH (22) et HTTP (80).

  14. Cliquez sur le bouton Vérifier + créer puis sur Créer. Il vous sera demandé de télécharger la clé privée. Acceptez et enregistrez-la dans un dossier de votre choix.

3 Connexion à la machine virtuelle

  1. Cliquez sur Accéder à la ressource pour afficher les détails de votre machine virtuelle.

  2. Copiez l’adresse IP publique.

  3. Entrez la commande suivante pour modifier les permissions de votre clé privée. La clé privée est stockée dans un fichier .pem que vous avez téléchargé précédemment.

chmod 644 <chemin_vers_fichier_pem>
  1. Entrez la commande suivante pour vous connecter à votre machine virtuelle.
ssh -i <chemin_vers_pem> <nom_utilisateur>@<adresse_ip_vm>

4 Installation du serveur web Nginx

Entrez simplement les deux commandes suivantes dans le terminal de la machine virtuelle :
sudo apt-get -y update
sudo apt-get -y install nginx

Une fois l’installation terminée, ouvrez le navigateur web de votre choix et saisissez l’adresse IP de la machine virtuelle. Vous devriez voir la page d’accueil par défaut de NGINX.

5 Nettoyage des ressources

N’oubliez pas d’arrêter la machine virtuelle, sinon votre crédit sera consommé ! Comme nous n’utiliserons plus cette machine virtuelle, il est recommandé de supprimer l’ensemble du groupe de ressources créé lors de cette activité. Vous pouvez consulter à tout moment le crédit associé à votre abonnement en cliquant sur ce lien.
